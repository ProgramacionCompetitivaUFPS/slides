String vs. StringBuilder

String vs. StringBuilder

Java provee dos maneras de manejar cadenas de texto: String y StringBuilder (o StringBuffer).

String

  • Manejo como tipo de dato primitivo.
  • Permite inicializar con: String s = "Hola";
  • Permite concatenaciones con el signo "+"
  • Internamente, cada concatenación crea un nuevo objeto.

StringBuilder

  • Manejo como tipo objeto.
  • Para inicializar: StringBuilder s = new StringBuilder();
  • Las concatenaciones se realizan con el método append();
  • Internamente aumenta el tamaño dinamicamente sin crear nuevos objetos.

Manipulación de String

Hay ciertos problemas de String que no requieren el uso de algoritmos o estructuras de datos complejas para el manejo de String, tan solo hay que idear una manera correcta y eficiente de hacer lo que pide el problema.

Manipulación de String

Para este tipo de problemas es muy importante tener un buen manejo del lenguaje de programación y conocer metodos que faciliten la codificación.

Manipulación de String

Algunos tipos de problemas de manipulación de String clasicos son:

  • Codificar/Decodificar un mensaje.
  • Frecuencia de una letra o una palabra.
  • Modificar los datos de entrada.
  • Formatear los datos de salida (leer Format).
  • Comparar String.
  • Otros...

Manipulación de String

Algunos ejemplos de métodos utiles de StringBuilder son:

  • append(...): Permite concatenar datos de cualquier tipo primitivo (o incluso un char[]) al final del String.
  • charAt(i): Retorna el caracter de la posición i.
  • getChars(i, j, arr, k): Guarda en el char[] arr a partir de la posición k, los caracteres de la posición i hasta la j-1.
  • reverse(): Invierte el contenido del StringBuilder.
  • substring(i, j): Retorna un String que contiene el substring de la posición i hasta la j-1.

Z-Function

Z-Function

Definición: "El Arreglo-Z de un String s de largo n contiene para cada i = 0, 1, ..., n-1 el largo de el substring más largo de s que inicia en la posición i y es un prefijo de s." (Laaksonen, A. Competitive Programmer’s Handbook).

Z-Function

Dado un String s retorna un arreglo z donde z[i] es igual al mayor numero de caracteres desde s[i] que coinciden con los caracteres desde s[0].

Dado un Texto t y un Patrón p, el problema es: Buscar todas las ocurrencias del patrón p en el texto t.
